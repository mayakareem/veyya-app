## Key Learnings

### For First-Time Users
1. **Social proof is entry point** - Friend referral + reviews
2. **Trust must be earned quickly** - Verification, guarantees, reviews
3. **Simplicity wins** - Each friction point loses users
4. **First service is audition** - Quality determines retention
5. **Second booking is critical** - Much easier, tests retention

### For Product Development
- Streamline onboarding (< 3 min target)
- Reviews front and center
- First-time anxiety addressed explicitly
- Proactive communication throughout
- Post-service follow-up drives reviews
- Incentivize second booking
